# Widgets Dashboard

- [ ] Add an editable widgets dashboard.
- [ ] Configurable by the user to:
  - Add widgets
  - Remove widgets
  - Reorder widgets

## What needs to be done

### 1. UI Infrastructure

- [ ] Implement a grid-based layout that supports draggable and resizable components.
- [ ] Create a "Widget Gallery" where users can pick which widgets to add.

### 2. Widget Development

- [ ] **Macro Summary Widget**: Visual breakdown of P/C/F.
- [ ] **Hydration Widget**: Quick track for water intake.
- [ ] **Weight Trend Widget**: Sparkline of recent weight logs.

### 3. Persistence

- [ ] Save the user's dashboard configuration (widget IDs, positions) in the database.
